# Home Lab - Basic Network Configuration

## Objective

Create a simple home network and verify connectivity between devices.

---

## Environment

- Windows 11 PC
- Home Router
- Ethernet Connection
- Internet Access

---

## Network Diagram

```
Internet
    │
 Router
    │
Windows PC
```

---

## Commands Used

```cmd
ipconfig
```

```cmd
ping 8.8.8.8
```

```cmd
ping google.com
```

```cmd
ipconfig /all
```

---

## Verification

- Internet working
- DNS resolution successful
- IP Address assigned by DHCP
- Gateway reachable

---

## Skills Demonstrated

- Windows Networking
- DHCP
- DNS
- TCP/IP
- Ping
- IPConfig
