# SSH (Secure Shell)

## Objective

Learn how to securely connect to remote Linux systems using SSH.

---

## What is SSH?

SSH (Secure Shell) is a secure network protocol used to remotely access and manage Linux servers.

---

## Basic SSH Command

```bash
ssh username@ip-address
```

Example:

```bash
ssh ahmed@192.168.1.10
```

---

## Useful SSH Commands

Check SSH version:

```bash
ssh -V
```

Exit SSH session:

```bash
exit
```

---

## Common Uses

- Remote server administration
- Managing Linux machines
- Secure file transfer (SCP)
- System troubleshooting
