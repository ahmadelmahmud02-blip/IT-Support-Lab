# DNS (Domain Name System)

## Objective

Understand how DNS translates domain names into IP addresses.

---

## What is DNS?

DNS (Domain Name System) converts domain names such as google.com into IP addresses that computers use to communicate.

Example:

google.com → 142.250.xxx.xxx

---

## Why is DNS Important?

- Makes websites easy to access.
- Eliminates the need to remember IP addresses.
- Essential for Internet communication.

---

## Common DNS Servers

Google DNS

```
8.8.8.8
8.8.4.4
```

Cloudflare DNS

```
1.1.1.1
1.0.0.1
```

---

## Useful Commands

Display DNS information

```cmd
ipconfig /all
```

Clear DNS Cache

```cmd
ipconfig /flushdns
```

Test DNS

```cmd
nslookup google.com
```
