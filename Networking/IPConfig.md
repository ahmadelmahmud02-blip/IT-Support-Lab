# IPConfig Command

## Objective

Learn how to use the ipconfig command for network troubleshooting on Windows.

---

## What is ipconfig?

The `ipconfig` command displays and manages the IP configuration of a Windows computer.

---

## Common Commands

Display IP configuration

```cmd
ipconfig
```

Detailed information

```cmd
ipconfig /all
```

Release IP Address

```cmd
ipconfig /release
```

Renew IP Address

```cmd
ipconfig /renew
```

Flush DNS Cache

```cmd
ipconfig /flushdns
```

---

## Practical Uses

- Check IP address
- Verify Default Gateway
- View DNS Servers
- Renew DHCP lease
- Troubleshoot Internet connectivity
