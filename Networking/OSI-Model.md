# OSI Model

## Objective

Understand the seven layers of the OSI networking model.

---

## OSI Layers

7. Application

6. Presentation

5. Session

4. Transport

3. Network

2. Data Link

1. Physical

---

## Why is it Important?

The OSI model helps troubleshoot network problems by identifying where communication issues occur.
