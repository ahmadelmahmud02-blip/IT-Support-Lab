# Ping

## Objective

Use the ping command to test network connectivity.

---

## What is Ping?

Ping sends ICMP echo requests to another device to verify connectivity and measure response time.

---

## Basic Command

```cmd
ping google.com
```

Ping an IP address

```cmd
ping 8.8.8.8
```

Continuous Ping

```cmd
ping -t google.com
```

Stop with:

```
Ctrl + C
```

---

## Common Uses

- Check Internet connectivity
- Test communication between devices
- Measure latency
- Troubleshoot network issues
