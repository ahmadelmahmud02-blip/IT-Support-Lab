# Remote Desktop (RDP)

## Objective

Learn how to remotely access Windows computers using Remote Desktop.

---

## What is Remote Desktop?

Remote Desktop Protocol (RDP) allows you to connect to another Windows computer remotely.

---

## Requirements

- Windows Pro or Enterprise
- Remote Desktop enabled
- Computer connected to the network
- Username and password

---

## Steps

1. Enable Remote Desktop.
2. Find the computer IP address.
3. Open Remote Desktop Connection.
4. Enter the IP address.
5. Sign in with user credentials.

---

## Common Uses

- Remote IT Support
- System Administration
- Remote Troubleshooting

---

## Skills Practiced

- Remote Desktop
