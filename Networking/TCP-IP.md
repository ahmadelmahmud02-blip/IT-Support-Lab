# TCP/IP

## Objective

Understand the TCP/IP protocol suite and its role in computer networking.

---

## What is TCP/IP?

TCP/IP (Transmission Control Protocol / Internet Protocol) is the standard communication protocol used by devices on the Internet and local networks.

---

## TCP

Transmission Control Protocol (TCP)

Features:

- Reliable communication
- Error checking
- Data retransmission
- Connection-oriented

Examples:

- HTTP
- HTTPS
- FTP
- SSH

---

## IP

Internet Protocol (IP)

Responsibilities:

- Identifies devices using IP addresses
- Routes packets between networks

Example IPv4 Address:

```
192.168.1.100
```

---

## IPv4 vs IPv6

| IPv4 | IPv6 |
|-------|-------|
| 32-bit | 128-bit |
| 192.168.1.1 | 2001:db8::1 |

---

## Useful Commands

Show IP configuration

```cmd
ipconfig
```

Detailed information

```cmd
ipconfig /all
```

Test connectivity

```cmd
ping 8.8.8.8
```
