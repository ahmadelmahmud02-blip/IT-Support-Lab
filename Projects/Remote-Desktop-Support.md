# Project 4 - Remote Desktop Support

## Objective

Provide remote technical support using Microsoft Remote Desktop.

---

## Requirements

- Windows Pro
- Remote Desktop Enabled
- Network Connectivity

---

## Steps

1. Enable Remote Desktop.
2. Obtain the remote computer IP address.
3. Launch Remote Desktop Connection.
4. Enter the IP address.
5. Authenticate using valid credentials.

---

## Skills Demonstrated

- Remote Desktop
- Windows Administration
- IT Support
- Remote Troubleshooting
