# Project 3 - SSH Connection to Linux

## Objective

Connect securely to a Linux machine using SSH.

---

## Environment

- Windows 11
- Kali Linux
- OpenSSH Client

---

## Command

```bash
ssh username@192.168.1.10
```

---

## Steps

1. Verify the Linux machine is online.
2. Check the IP address.
3. Open Command Prompt or PowerShell.
4. Run the SSH command.
5. Enter the password.
6. Verify successful login.

---

- SSH
- Linux
