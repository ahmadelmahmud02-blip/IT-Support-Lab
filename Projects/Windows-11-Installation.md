# Project 2 - Windows 11 Installation

## Objective

Install Windows 11 on a new computer and prepare it for daily use.

---

## Requirements

- Windows 11 ISO
- USB Flash Drive (8GB or larger)
- Rufus
- Internet Connection

---

## Installation Steps

1. Download the Windows 11 ISO.
2. Create a bootable USB using Rufus.
3. Boot the computer from the USB drive.
4. Install Windows 11.
5. Complete the initial setup.
6. Install motherboard drivers.
7. Install graphics drivers.
8. Install Windows Updates.
9. Install essential software.

---

## Verification

- Windows activated
- Drivers installed successfully
- Internet connection working
- Windows Update completed

---

## Skills Demonstrated

- Windows Installation
- BIOS Boot Configuration
- Driver Installation
- Windows Update
- Hardware Setup
