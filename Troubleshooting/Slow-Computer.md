# Slow Computer Troubleshooting

## Problem

The computer is running very slowly.

---

## Possible Causes

- High CPU usage
- Low RAM
- Full storage
- Startup applications
- Malware

---

## Troubleshooting Steps

1. Open Task Manager.
2. Check CPU, RAM and Disk usage.
3. Disable unnecessary startup programs.
4. Free disk space.
5. Scan for malware.
6. Restart the computer.

---

## Tools Used

- Task Manager
- Disk Cleanup
- Windows Security
