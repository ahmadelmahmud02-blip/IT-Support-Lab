# Driver Installation

## Objective

Install and verify hardware drivers after Windows installation.

---

## Drivers Installed

- Motherboard (Chipset)
- Graphics Card (NVIDIA / AMD)
- Network Adapter
- Audio Driver
- Bluetooth Driver
- Wi-Fi Driver (if available)

---

## Installation Process

1. Identify the hardware model.
2. Download the latest drivers from the official manufacturer.
3. Install chipset drivers first.
4. Install graphics drivers.
5. Install network and audio drivers.
6. Restart the computer.
7. Verify installation using Device Manager.

---

## Verification

- No unknown devices.
- No warning icons in Device Manager.
- Internet connection working.
- Audio functioning correctly.
- Display resolution configured properly.

---

## Best Practices

- Download drivers only from official websites.
- Install chipset drivers before other drivers.
- Restart the computer after major driver installations.
