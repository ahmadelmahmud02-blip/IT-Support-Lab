# Windows Installation

## Objective

Install Windows 10/11 and prepare the computer for daily use.

---

## Requirements

- Windows ISO
- USB Flash Drive (8GB or larger)
- Rufus
- Device Drivers

---

## Installation Steps

1. Download the Windows ISO.
2. Create a bootable USB using Rufus.
3. Boot the computer from the USB drive.
4. Install Windows.
5. Install motherboard drivers.
6. Install graphics card drivers.
7. Run Windows Update.
8. Install essential software.

---

## Verification

- Windows activated successfully.
- Drivers installed correctly.
- Windows Update completed.
- Internet connection working.

---
